# Tianguis Tycoon 🏪✨

¡Bienvenido al simulador de tianguis definitivo! Un juego de regateo y estrategia con temática de mercado mexicano.

## 🚀 Cómo Jugar
1. **Atiende a los clientes**: Haz clic en los NPCs que se acerquen a tu puesto.
2. **Negocia**: Usa el panel de regateo para ofrecer precios justos. ¡Cuidado con su paciencia!
3. **Mejora tu puesto**: Junta dinero para subir de nivel y atraer mejores objetos.

## 🛠️ Tecnologías
- **Phaser 3**: Motor de juego.
- **Vanilla JavaScript**: Lógica y sistemas.
- **HTML5/CSS3**: Interfaz y transiciones.
